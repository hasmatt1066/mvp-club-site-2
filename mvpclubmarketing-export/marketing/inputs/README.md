# Inputs

Drop raw content here for Claude to process.

## Subfolders

- `/transcripts/` - Session transcripts, call recordings
- `/recordings/` - Raw audio/video files
- `/ideas/` - Founder notes, voice memos, rough ideas
- `/campaigns/` - Campaign briefs, announcements

## Naming Convention

```
YYYY-MM-DD_type_description.md

Examples:
2025-01-15_transcript_demo-day-claude-code.md
2025-01-15_idea_managerial-relationship-post.md
2025-01-15_campaign_founding-members.md
```
